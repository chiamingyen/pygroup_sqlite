Creating detail entities

This example function (“lineEntityCreate()”) demonstrates creation of detail entities.  The current model must be a drawing.  After selecting on the button below choose the endpoints of the line with the mouse.

 

                                                                                                                            

Creating detail notes

This example function (“createDetailNote()”) demonstrates creation of detail notes.  The current model must be a drawing.  The note text will be the name or id of the selected surface, and the note text will be placed at the upper right of the drawing view outline.
