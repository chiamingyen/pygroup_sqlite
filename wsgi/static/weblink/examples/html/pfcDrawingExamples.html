Creating drawings from templates

This example function (“createDrawingFromTemplate()”) demonstrates creation of drawings using Pro/Web.Link.  The drawing template used is hardcoded (“c_drawing”) in the function and may be modified as needed.  The name of the new drawing should be supplied in the input panel below.  A solid model should be current model.

 

Listing drawing sheets

This example function ("listSheets()") demonstrates the use of methods in the pfcSheetOwner class. A drawing must be the Pro/ENGINEER current model for this example.

 

 

Listing drawing views

This example function (“listViews()”) demonstrates the use of the properties and methods in the pfcView2D class.  A drawing must the Pro/ENGINEER current model for this example.

 

 

Replacing drawing models

This example function (“replaceModels()”) demonstrates access to drawing models.  Any solid model instances will be replaced by their generic.  Similar to the Pro/ENGINEER behavior,  the function will not replace models if the target generic model is already present in the drawing.
