Modifying dimension tolerances

 

This example function (“setAngularToleranceToLimits”) sets angular tolerances to a specified range.   For each angular dimension in the current model the function gets the dimension value and adds or subtracts the range to it to get the upper and lower limits.  The function then initializes a pfcDimTolLimits tolerance object and assigns it to the dimension.   The function displays each shown dimension.  The current model must be a part or assembly.
