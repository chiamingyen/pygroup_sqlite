Pro/Web.Link Component Feature Examples

 

Replacing components

 

This example function (“replaceBoltsInAssembly”) replaces one instance of a component with another in an assembly.  The current model must be an assembly.  It should contain a subcomponent “bolt.prt” with instances “slot7_8” and “phillips7_8”.

 

 

 

Accessing component assembly constraints

 

This example function (“highlightConstraints”) highlights each of the pairs of constraints for a selected component visually, and lists the type and offset values for the component.  The current model must be an assembly.  Select a component feature from the window to show its constraints.

 

 

 

Assembling components

 

This example function (“assembleByDatums”) auto-assembles the component entered below by aligning datums. Enter the component name as ??????.prt or ?????.asm.  The current model must be an assembly.

The required datum pairs are

Assembly datum

Component Datum

ASM_D_FRONT

COMP_D_FRONT

ASM_D_TOP

COMP_D_TOP

ASM_D_RIGHT

COMP_D_RIGHT

 

If any of the required datums are not found the component will be underconstrained and the Component placement UI will come up so the problem can be corrected.
