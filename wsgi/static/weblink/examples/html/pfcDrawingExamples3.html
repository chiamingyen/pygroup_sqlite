Placing symbol instances

This example function (“placeSymInst()”) demonstrates placement of symbol instances.  .  The current model must be a drawing.   Select one or more locations for the symbol instance (terminate selection with the middle mouse button).

 

 

 

Creating a detail group

This example function (“createGroup()”) demonstrates creation of a detail group.  The current model must be a drawing.  Enter the name of the group in the input box below.  Select notes, entities and symbols to add to the detail group.

 

 

Creating a table of point information

This example function (“createTableOfPoints()”) demonstrates creation of a table.  Select a coordinate system from a model in a drawing view.  Geometric information on all the points in the top level model will be written in the new table.
