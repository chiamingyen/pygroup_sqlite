Pro/Web.Link Selection Examples

 

Prompting for selection

 

This example function (“selectItems”) demonstrates interactive selection.   Choose one or more strings to use for selection item types, and enter the maximum number of selections.

 

 

 

Programmatic selection

 

This example function (“createAndHighlightSelections”) demonstrates programmatic selection.   Enter the name of feature(s) found in the current assembly.  This function traverses the assembly and highlights each feature with the given name.
