Library | Module

Enum pfcServerObjectLockStatus

Description

Describes the type of lock applied to an object on the server.



Values

OBJ_LOCK_STAT_UNSET
No lock is set.
OBJ_LOCK_STAT_UNLOCKED
The object is unlocked.
OBJ_LOCK_STAT_HARDLOCK
The object has a hard lock.
OBJ_LOCK_STAT_SOFTLOCK
The object has a soft lock.
ServerObjectLockStatus_nil
Use this enumerated value to represent "null" passed to optional properties or method arguments.
